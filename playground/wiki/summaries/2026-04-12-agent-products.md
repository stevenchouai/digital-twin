# Agent Products Summary

## Core Judgment

大多数 AI 产品的问题不是 capability 不够，而是工作结果不够 durable。

## Supporting Points

- 用户每次都要重新解释上下文
- 很多系统没有 write-back，所以不会积累
- 没有 learning loop 的 agent 很难变得越来越贴近用户

## Reusable Language

- `answer machine`
- `asset-building machine`
- `work trace`
- `compounding workflow`
